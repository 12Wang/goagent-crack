0.使用推荐账号先翻墙
goagent-goagent-985cbd5为已经配置好的goagent包，可以跳到第4步先翻墙，完成以下操作后更新为自己appid
1.申请gae appid，具体操作可google
https://console.developers.google.com/project?authuser=3
2.使用appid上传goagent server到google appengine，具体操作可google
需要开启gae密码弱应用 https://www.google.com/settings/security/lesssecureapps
3.配置goagent local中的appid/google ip(proxy.ini)
google ip采用推荐的ip
4.启动goagent local
5.安装chrome插件SwitchySharp.crx并配置本地代理
